Nodemon
Tools pertama adalah nodemon, ia bisa dikatakan wajib digunakan selama proses pengembangan. Pasalnya, dengan tools ini kita tak perlu menjalankan ulang server ketika terjadi perubahan pada berkas JavaScript. 
Nodemon akan mendeteksi perubahan kode JavaScript dan mengeksekusi ulang secara otomatis.

-> npm install nodemon --save-dev
